Instalación y Configuración de Linux

Servicio a domicilio de instalación de Linux para computadoras de empresas, casas, oficina y usuarios en general.

Solo como sistema operativo único. NO dual boot, no se instala en una computadora con Windows, salvo que se formatee y realice la instalación de Linux como sistema único.

El sistema operativo es indicado para el usuario promedio que desee:

  • Navegar por internet.
  • Ver Netflix, Youtube, u otros servicio de video streaming.
  • Pagar online, Banca por Internet.
  • Crear, editar documentos (Textos, Hoja de Cálculos), compatibilidad con doc, docx, xls, etc.
  • Convertir fácilmente cualquier documento a PDF.
  • Grabar una imagen ISO a un DVD.
  • Editar sus fotos o imágenes.
  • Crear y editar diseños vectoriales.
  • Editar video y audio, capas, máscaras, efectos, etc.
  • Conversión de formatos de video, imagen, audio y documentos.
  • Usar un cliente de correo para manejar todas sus cuentas desde un mismo aplicativo.
  • Empleo de Cortafuegos de manera sencilla.
